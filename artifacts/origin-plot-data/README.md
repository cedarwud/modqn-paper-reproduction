# Origin Plot Data

This directory is reserved for curated OriginLab-ready exports.

Use it for small, reviewable plotting inputs derived from completed
runs or sweeps, for example:

1. `table-ii.csv`
2. `fig-3-detail.csv`
3. `fig-3-weighted-reward.csv`
4. `fig-4-detail.csv`
5. `fig-4-weighted-reward.csv`
6. `fig-5-detail.csv`
7. `fig-5-weighted-reward.csv`
8. `fig-6-detail.csv`
9. `fig-6-weighted-reward.csv`

Do not place checkpoints, full training logs, replay timelines, or raw
run bundles here. Those remain generated artifacts rather than tracked
authority surfaces.

Regeneration commands and episode choices are documented in:

1. `docs/origin-plot-data-runbook.md`
